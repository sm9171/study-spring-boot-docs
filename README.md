## study-spring-boot-docs

### 주제 : 스프링 공식문서 스터디  
### 공부거리 : [스프링 부트 2.6.11 공식문서](https://docs.spring.io/spring-boot/docs/2.6.11-SNAPSHOT/reference/htmlsingle/)
### 진행 방식
1. 스터디 장이 매주 공부할 분량을 정해 이슈 발행한다.
2. 정해진 분량을 학습 후 정리한 내용이나 실습 코드 PR 요청으로 인증한다.
---

### PR 메세지 규칙
1. 제목 규칙 : 이름_차수  
  ex) 홍길동_1주차
2. 내용 규칙 : 
  - 이슈 번호를 반드시 기입한다
